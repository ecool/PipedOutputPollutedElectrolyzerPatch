# PipedOutputPollutedElectrolyzerPatch

A patch to add `PipedOutput` to `PollutedElectrolyzer`.

# How To Use
* This mod needs to be loaded after `PipedOutput` and `PollutedElectrolyzer` in the Mods Page.
